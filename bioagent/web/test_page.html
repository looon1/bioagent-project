API 端点

  • GET /api/health - 健康检查
  • GET /api/sessions - 列出所有会话
  • POST /api/sessions - 创建新会话
  • POST /api/chat - SSE 流式聊天
  • POST /api/upload - 文件上传
  • GET /api/files?path=... - 下载文件
  • GET /api/dir?path=... - 列出目录

使用方法

CLI 启动:

bioagent-web --port 7860

Python API:

from bioagent.web import create_app
from bioagent.agent import Agent
from bioagent.config import BioAgentConfig

config = BioAgentConfig.from_env()
agent = Agent(config=config)
app = create_app(agent, config)
uvicorn.run(app, host="0.0.0.0", port=7860)
